# 제작 과정

## 1단계 - 브랜드 기획

생각 정리 AI 서비스인 MindFlow를 가상 브랜드로 선정하였다.

타겟은 정보 과부하를 경험하는 직장인으로 설정하였다.

핵심 메시지는 "흩어진 생각이 흐름이 되는 순간"으로 정의하였다.

---

## 2단계 - 스토리보드 설계

광고 구조를 다음과 같이 설계하였다.

1. 정보 과부하
2. 업무 마비
3. MindFlow 등장
4. 생산성 향상
5. 브랜드 엔딩

문제 → 해결 → 성과 구조를 사용하여 메시지를 전달하였다.

---

# 제작 중 발생한 문제

## 문제 1

씬마다 인물 얼굴이 변경되는 현상 발생

### 원인

텍스트 기반 생성만 사용하여 캐릭터 일관성이 유지되지 않음

### 해결

Scene 1 최종 이미지를 Character Reference로 활용하였다.

동일 인물, 동일 의상, 동일 헤어스타일을 유지하도록 프롬프트를 수정하였다.

---

## 문제 2

영상 생성 시 UI와 텍스트가 깨지는 현상 발생

### 원인

영상 생성 모델이 작은 텍스트와 복잡한 UI를 안정적으로 유지하지 못함

### 해결

UI 변화가 많은 Scene 2와 Scene 3은 정지 이미지 기반 연출을 사용하였다.

Zoom, Fade 효과를 활용하여 영상 흐름을 유지하였다.

최종 광고는 정보 과부하를 경험하는 직장인이 MindFlow를 통해 업무를 정리하고 생산성을 회복하는 과정을 표현하였다.

AI 이미지 생성, AI 영상 생성, AI 음악 생성, 온라인 편집 도구를 활용하여 하나의 통합된 브랜드 광고를 제작하였다.

# 회고

이번 프로젝트를 통해 단순히 이미지를 생성하는 것이 아니라,

기획 → 프롬프트 설계 → 생성 → 수정 → 편집

으로 이어지는 AI 콘텐츠 제작 파이프라인의 중요성을 경험할 수 있었다.

특히 캐릭터 일관성 유지와 영상 생성 시 발생하는 UI 왜곡 문제를 해결하는 과정에서 프롬프트 설계와 제작 전략의 중요성을 확인할 수 있었다.
