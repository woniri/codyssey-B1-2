# 브랜드 아이덴티티 및 캠페인 개요

## 브랜드명

MindFlow

## 브랜드 소개

MindFlow는 AI 기반 생산성 플랫폼으로, 사용자의 흩어진 생각과 업무 정보를 자동으로 정리하여 실행 가능한 계획으로 전환해주는 서비스이다.

## 타겟 고객

* 20~40대 직장인
* 다양한 업무와 회의, 이메일, 메신저를 동시에 관리해야 하는 지식 노동자
* 정보 과부하로 인해 집중력 저하와 업무 우선순위 설정에 어려움을 겪는 사용자

## 문제 정의

현대의 업무 환경에서는 이메일, 메신저, 일정, 회의 내용, 아이디어 메모 등 수많은 정보가 동시에 유입된다.

이로 인해 사용자는 중요한 업무를 놓치거나 우선순위를 판단하기 어려워진다.

## 핵심 기능

MindFlow는 흩어진 업무 정보와 생각을 AI가 자동으로 분석하여 우선순위를 정리하고 실행 가능한 작업 목록으로 변환한다.

## USP (차별화 포인트)

단순한 정보 요약이 아니라, 사용자가 즉시 실행할 수 있는 우선순위와 액션 플랜까지 자동으로 제안한다.

## 톤앤매너

* 전문적
* 미래지향적
* 신뢰감
* 깔끔하고 직관적인 경험

## 비주얼 스타일

* 실사 기반 광고
* 프리미엄 시네마틱 스타일
* 블루 계열 미래형 UI
* 미니멀하고 세련된 연출

## 캠페인 목표

브랜드 인지도 향상 (Brand Awareness)
