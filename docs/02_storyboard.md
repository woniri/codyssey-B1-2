# MindFlow 광고 스토리보드

## 프로젝트 개요

* 브랜드명: MindFlow
* 캠페인 목표: 브랜드 인지도 향상 (Brand Awareness)
* 핵심 메시지: "흩어진 생각이 흐름이 되는 순간."
* 타겟: 정보 과부하를 겪는 직장인
* 총 영상 길이: 약 45초

---

# Scene 1

## 씬 번호

Scene 1

## 씬 길이

8초

## 목표 메시지

너무 많은 정보가 집중을 방해한다.

## 화면 구성

* 현대적인 사무실
* 이메일, 메신저, 일정 알림이 화면을 가득 채움
* 스트레스를 받는 직장인

## 내레이션 또는 화면 카피

"생각보다 정보가 먼저 쌓인다."

## 사용 도구

* ChatGPT Image Generation
* Veo
* CapCut

## 사용 목적

정보 과부하 상황 표현

## 입력 프롬프트

Prompt V2 사용

## 출력 결과 요약

이메일, 메신저, 일정 알림이 동시에 몰려오는 정보 과부하 상황을 표현하였다. 직장인의 스트레스와 업무 압박감을 시각적으로 전달한다.

# 스타일 일관성 유지 전략

Scene 1 최종 이미지(MF_S01_v2.png)를 기준 캐릭터로 설정하였다.

이후 모든 씬 생성 시 Character Reference 방식으로 동일 인물, 동일 헤어스타일, 동일 의상을 유지하였다.

공통 스타일 키워드:

* Photorealistic
* Premium Commercial Advertising
* Cinematic Lighting
* Blue Futuristic UI
* Modern Office Environment
* 16:9 Aspect Ratio

# 영상 생성 전략

* Scene 1 : Veo 영상 생성
* Scene 2 : 정지 이미지 + CapCut Zoom
* Scene 3 : Veo 영상 생성
