# MindFlow 광고 스타일 가이드

## 스타일 목표

MindFlow는 복잡한 정보를 정리하여 생산성을 향상시키는 AI 서비스이다.

광고 전반에서 전문성, 신뢰성, 집중력, 미래지향적 이미지를 전달하는 것을 목표로 한다.
